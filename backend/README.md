Backend server code for calculating actual routes and passing data to end users


Main language - Python


main.py - used in early stage of development, hosts a library test and graph approach attempt, currently unused and waiting for reassignment

init.py - all functions for transforming gtfs and network data into standardised structures defined in "structs.py"

structs.py - definition of all classes and structs related to data shaping and storing for easy access down the line
